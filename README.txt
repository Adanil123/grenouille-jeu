# Projet Grenouille Unity 3D

## Instructions

- Ouvre ce projet avec Unity (2020+ recommandé).
- Crée une nouvelle scène.
- Ajoute un **Plane** comme sol. Mets-lui le tag `Ground`.
- Crée un **GameObject vide** nommé `Grenouille`.
- Ajoute sur ce GameObject :
  - Composants : `MeshFilter`, `MeshRenderer`, `Rigidbody`, `CapsuleCollider`
  - Script : `SimpleFrogMesh`
  - Script : `FrogController`
- Assure-toi que le Rigidbody a `Use Gravity` activé.
- Lance la scène, contrôle la grenouille avec les flèches ou ZQSD, et espace pour sauter.

---

Ce projet est un exemple simple avec une grenouille low-poly générée en code.
